    fmod-studio-docs

    Project maintained by bobthenameless Hosted on GitHub Pages — Theme by mattgraham

    This is a project to strip the FMOD Studio API for Windows documentation out of the .chm file and into a web accessible format. FMOD can be downloaded/purchased at http://www.fmod.org/ and the .chm file can be found in API_INSTALL_DIRECTORY/docs. This version has been done with a mix of manual and automatic tinkering, and in the future i will be attempting to provide a way of more fully automating the process.

    The .chm file also contains additional sections aside from what is covered here. This page is mainly concerned with the API reference section of the .chm, but the additional sections in the .chm include an introduction, overview, and examples. There's good information to be had there, but currently there is no plan to convert these sections to this site.
