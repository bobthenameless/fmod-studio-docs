ChannelControl::getReverbProperties

Retrieves the wet level (or send level) for a particular reverb instance.

C++ Syntax

FMOD_RESULT ChannelControl::getReverbProperties(
  int instance,
  float *wet
);

Parameters

instance

Index of the particular reverb instance to target, from 0 to 3 inclusive.

wet

Address of a variable that receives the send level for the signal to the reverb, from 0 (none) to 1.0 (full).

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
