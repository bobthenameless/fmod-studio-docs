Firelight Technologies FMOD Studio API

FSBank_Release

Release the FSBank system, clean up used resources.

C Syntax

FSBANK_RESULT FSBank_Release();

Return Values

If the function succeeds then the return value is FSBANK_OK.
If the function fails then the return value will be one of the values defined in the FSBANK_RESULT enumeration.

Remarks

All progress items retrieved with FSBank_FetchNextProgressItem will be released by this function.
