FMOD_STUDIO_RECORD_COMMANDS_FLAGS

Flags passed into Studio::System::startRecordCommands.

Definition

#define FMOD_STUDIO_RECORD_COMMANDS_NORMAL 0x00000000
#define FMOD_STUDIO_RECORD_COMMANDS_FILEFLUSH 0x00000001

Values

FMOD_STUDIO_RECORD_COMMANDS_NORMAL

Standard behaviour.

FMOD_STUDIO_RECORD_COMMANDS_FILEFLUSH

Call file flush on every command.
