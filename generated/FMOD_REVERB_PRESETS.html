FMOD_REVERB_PRESETS

A set of predefined environment PARAMETERS.
These are used to initialize an FMOD_REVERB_PROPERTIES structure statically.
i.e.
FMOD_REVERB_PROPERTIES prop = FMOD_PRESET_GENERIC;
