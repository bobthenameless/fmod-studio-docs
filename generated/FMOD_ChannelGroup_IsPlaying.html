ChannelGroup::isPlaying

Retrieves the playing state.

C++ Syntax

FMOD_RESULT ChannelGroup::isPlaying(
  bool *isplaying
);

C Syntax

FMOD_RESULT FMOD_ChannelGroup_IsPlaying(
  FMOD_CHANNELGROUP *channelgroup,
  FMOD_BOOL *isplaying
);

Parameters

isplaying
Address of a variable that receives the current playing status, true = currently playing a sound, false = not playing a sound.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
