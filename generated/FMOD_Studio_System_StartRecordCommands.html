Firelight Technologies FMOD Studio API

Studio::System::startRecordCommands

Start recording all Studio commands to a file with the given path.

C++ Syntax

FMOD_RESULT Studio::System::startRecordCommands(
  const char *filename,
  FMOD_STUDIO_RECORD_COMMANDS_FLAGS flags
);

C Syntax

FMOD_RESULT FMOD_Studio_System_StartRecordCommands(
  FMOD_STUDIO_SYSTEM *system,
  const char *filename,
  FMOD_STUDIO_RECORD_COMMANDS_FLAGS flags
);

Parameters

filename
flags

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
