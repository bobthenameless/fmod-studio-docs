Firelight Technologies FMOD Studio API

Studio::MixerStrip::setFaderLevel

Sets the volume of the MixerStrip.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::setFaderLevel(
  float level
);

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_SetFaderLevel(
  FMOD_STUDIO_MIXERSTRIP *mixerstrip,
  float level
);

Parameters

level
The volume level to set as a linear gain. 0 = silent, 1 = full volume.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

This function allows volume control of all audio routed into the mixer strip. This fader level is relative to the fader level set in FMOD Studio.
