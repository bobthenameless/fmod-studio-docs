Firelight Technologies FMOD Studio API

What's New in 1.01

This section describes the major features introduced in the 1.01 release. See the Detailed Revision History for information regarding each patch release.

Sends

Support has been added for Send DSP connections to send sound data to different areas of the mix graph.

Fade Points

Support has been added for setting sample-accurate fade ramp points. The fade points allow Studio to schedule sound volume gradients to occur in future mixes.

Studio API

A number of additional Studio API functions have been added. See the revision.txt for more details.
