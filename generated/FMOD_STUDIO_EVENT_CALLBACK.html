Firelight Technologies FMOD Studio API

FMOD_STUDIO_EVENT_CALLBACK

Callback that is fired when a Studio::EventInstance changes state.

C/C++ Syntax

FMOD_RESULT F_CALLBACK FMOD_STUDIO_EVENT_CALLBACK(
  FMOD_STUDIO_EVENT_CALLBACK_TYPE type,
  void *parameters
);

Parameters

type

The type of event that has occurred.

parameters

The callback parameters.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

This callback is used for tracking playback state and providing programmer sounds. The data passed to the callback function in the parameters argument varies based on the callback type. See FMOD_STUDIO_EVENT_CALLBACK_TYPE for more information.

Example:

FMOD_RESULT F_CALLBACK mycallback(FMOD_STUDIO_EVENT_CALLBACK_TYPE type, void *parameters)
{
    if (type == FMOD_STUDIO_EVENT_CALLBACK_STOPPED)
    {
        Studio::EventInstance *instance = (Studio::EventInstance *)parameters;

        // More code goes here.

    }

    return FMOD_OK;
}
