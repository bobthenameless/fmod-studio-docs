Firelight Technologies FMOD Studio API

Studio::MixerStrip::getID

Retrieves the ID of the mixer strip.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::getID(
  FMOD_GUID *id
);

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_GetID(
  FMOD_STUDIO_MIXERSTRIP *mixerstrip,
  FMOD_GUID *id
);

Parameters

id
Address of a variable to receive the 128-bit GUID.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
