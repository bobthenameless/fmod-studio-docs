Firelight Technologies FMOD Studio API

Studio::Bank::getEventCount

Retrieves the the number of EventDescriptions in the bank.

C++ Syntax

FMOD_RESULT Studio::Bank::getEventCount(
  int *count
);

C Syntax

FMOD_RESULT FMOD_Studio_Bank_GetEventCount(
  FMOD_STUDIO_BANK *bank,
  int *count
);

Parameters

count
Address of a variable to receive the number of EventDescriptions.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

Can be used in conjunction with Studio::Bank::getEventList to enumerate the events.
