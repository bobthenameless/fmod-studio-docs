ChannelControl::get3DDopplerLevel

Retrieves the amount by which doppler is scaled.

C++ Syntax

FMOD_RESULT ChannelControl::get3DDopplerLevel(
  float *level
);

Parameters

level

Address of a variable to receives the doppler scale from 0.0 (none), to 1.0 (normal) to 5.0 (exaggerated).

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
