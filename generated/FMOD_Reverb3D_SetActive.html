Reverb3D::setActive

Disables or enables a reverb object so that it does or does not contribute to the 3d scene.

C++ Syntax

FMOD_RESULT Reverb3D::setActive(
  bool active
);

C Syntax

FMOD_RESULT FMOD_Reverb3D_SetActive(
  FMOD_REVERB3D *reverb3d,
  FMOD_BOOL active
);

Parameters

active
true = active, false = not active. Default = true.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
