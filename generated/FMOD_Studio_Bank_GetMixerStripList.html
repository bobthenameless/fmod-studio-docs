Studio::Bank::getMixerStripList

Retrieves the Mixer Strips in the bank.

C++ Syntax

FMOD_RESULT Studio::Bank::getMixerStripList(
  Studio::MixerStrip **array,
  int capacity,
  int *count
);

C Syntax

FMOD_RESULT FMOD_Studio_Bank_GetMixerStripList(
  FMOD_STUDIO_BANK *bank,
  FMOD_STUDIO_MIXERSTRIP **array,
  int capacity,
  int *count
);

Parameters

array
An array of memory allocated by the user.
capacity
The capacity of the array passed in as the first parameter
count
Address of a variable to receive the number of Mixer Strips written to the array

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

Used in conjunction with Studio::Bank::getMixerStripCount to enumerate the Mixer Strips.
