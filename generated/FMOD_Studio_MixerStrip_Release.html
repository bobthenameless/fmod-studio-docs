Firelight Technologies FMOD Studio API

Studio::MixerStrip::release

Releases the MixerStrip handle.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::release();

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_Release(FMOD_STUDIO_MIXERSTRIP *mixerstrip);

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

All handles to this object are invalidated. Cancels any pending user-triggered loads of this object.
