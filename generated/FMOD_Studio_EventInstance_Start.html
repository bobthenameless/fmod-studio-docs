Firelight Technologies FMOD Studio API

Studio::EventInstance::start

Starts playback of the event instance.

C++ Syntax

FMOD_RESULT Studio::EventInstance::start();

C Syntax

FMOD_RESULT FMOD_Studio_EventInstance_Start(FMOD_STUDIO_EVENTINSTANCE *eventinstance);

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

This will begin playback of the event instance. If the instance was already playing it will restart the event.
