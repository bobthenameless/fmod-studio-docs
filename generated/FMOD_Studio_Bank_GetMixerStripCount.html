Firelight Technologies FMOD Studio API

Studio::Bank::getMixerStripCount

Retrieves the the number of Mixer Strips in the bank.

C++ Syntax

FMOD_RESULT Studio::Bank::getMixerStripCount(
  int *count
);

C Syntax

FMOD_RESULT FMOD_Studio_Bank_GetMixerStripCount(
  FMOD_STUDIO_BANK *bank,
  int *count
);

Parameters

count
Address of a variable to receive the number of Mixer Strips.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

Used in conjunction with Studio::Bank::getMixerStripList to enumerate the Mixer Strips.
