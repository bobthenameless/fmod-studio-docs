Firelight Technologies FMOD Studio API

Studio::MixerStrip::getPaused

Retrieves the pause state of the MixerStrip.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::getPaused(
  bool *paused
);

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_GetPaused(
  FMOD_STUDIO_MIXERSTRIP *mixerstrip,
  FMOD_BOOL *paused
);

Parameters

paused
Address of a variable to receive the pause state.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
