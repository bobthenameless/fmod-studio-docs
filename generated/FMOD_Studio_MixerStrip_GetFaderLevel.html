Studio::MixerStrip::getFaderLevel

Retrieves the volume level of the MixerStrip.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::getFaderLevel(
  float *level
);

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_GetFaderLevel(
  FMOD_STUDIO_MIXERSTRIP *mixerstrip,
  float *level
);

Parameters

level
Address of a variable to receive the volume level.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
