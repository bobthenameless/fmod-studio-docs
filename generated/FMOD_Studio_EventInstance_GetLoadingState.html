Studio::EventInstance::getLoadingState

This function is deprecated. Retrieves the loading state of the event instance.

C++ Syntax

FMOD_RESULT Studio::EventInstance::getLoadingState(
  FMOD_STUDIO_LOADING_STATE *state
);

C Syntax

FMOD_RESULT FMOD_Studio_EventInstance_GetLoadingState(
  FMOD_STUDIO_EVENTINSTANCE *eventinstance,
  FMOD_STUDIO_LOADING_STATE *state
);

Parameters

state
Address of a variable to receive the loading state.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.

Remarks

This function is deprecated. Use Studio::EventDescription::getSampleLoadingState instead.

You can use this function to check the loading state of the samples that are used by the event instance.
