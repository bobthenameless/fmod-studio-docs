Studio::MixerStrip::getLoadingState

Retrieves the loading state of the MixerStrip.

C++ Syntax

FMOD_RESULT Studio::MixerStrip::getLoadingState(
  FMOD_STUDIO_LOADING_STATE *state
);

C Syntax

FMOD_RESULT FMOD_Studio_MixerStrip_GetLoadingState(
  FMOD_STUDIO_MIXERSTRIP *mixerstrip,
  FMOD_STUDIO_LOADING_STATE *state
);

Parameters

state
Address of a variable to receive the loading state.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
