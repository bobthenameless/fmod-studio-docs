Firelight Technologies FMOD Studio API

ChannelGroup::stop

Stops the channel (or all channels in the channel group) from playing. Makes it available for re-use by the priority system.

C++ Syntax

FMOD_RESULT ChannelGroup::stop();

C Syntax

FMOD_RESULT FMOD_ChannelGroup_Stop(FMOD_CHANNELGROUP *channelgroup);

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
