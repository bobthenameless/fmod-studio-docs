Firelight Technologies FMOD Studio API

FMOD_DSP_PARAMETER_DATA_TYPE

Built-in types for the 'datatype' member of FMOD_DSP_PARAMETER_DESC_DATA. Data parameters of type other than FMOD_DSP_PARAMETER_DATA_TYPE_USER will be treated specially by the system.

Enumeration

typedef enum {
  FMOD_DSP_PARAMETER_DATA_TYPE_USER,
  FMOD_DSP_PARAMETER_DATA_TYPE_OVERALLGAIN,
  FMOD_DSP_PARAMETER_DATA_TYPE_3DATTRIBUTES,
  FMOD_DSP_PARAMETER_DATA_TYPE_SIDECHAIN
} FMOD_DSP_PARAMETER_DATA_TYPE;

Values

FMOD_DSP_PARAMETER_DATA_TYPE_USER

The default data type. All user data types should be 0 or above.

FMOD_DSP_PARAMETER_DATA_TYPE_OVERALLGAIN

The data type for FMOD_DSP_PARAMETER_OVERALLGAIN parameters. There should a maximum of one per DSP.

FMOD_DSP_PARAMETER_DATA_TYPE_3DATTRIBUTES

The data type for FMOD_DSP_PARAMETER_3DATTRIBUTES parameters. There should a maximum of one per DSP.

FMOD_DSP_PARAMETER_DATA_TYPE_SIDECHAIN

The data type for FMOD_DSP_PARAMETER_SIDECHAIN parameters. There should a maximum of one per DSP.
