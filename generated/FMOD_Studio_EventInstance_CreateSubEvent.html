Firelight Technologies FMOD Studio API

Studio::EventInstance::createSubEvent

Creates an instance of a subevent that is routed into the event instance.

C++ Syntax

FMOD_RESULT Studio::EventInstance::createSubEvent(
  const char *name,
  Studio::EventInstance **instance
);

C Syntax

FMOD_RESULT FMOD_Studio_EventInstance_CreateSubEvent(
  FMOD_STUDIO_EVENTINSTANCE *eventinstance,
  const char *name,
  FMOD_STUDIO_EVENTINSTANCE **instance
);

Parameters

name
Name of the subevent.
instance
Address of a variable to receive the new subevent instance object.

Return Values

If the function succeeds then the return value is FMOD_OK.
If the function fails then the return value will be one of the values defined in the FMOD_RESULT enumeration.
